func reduce<S : SequenceType, U>(sequence: S, initial: U, combine: (U, S.Generator.Element) -> U) -> U

Return the result of repeatedly calling combine with an accumulated value initialized to initial and each element of sequence, in turn.

For example, this function can be used to merge values:

var a = reduce([5, 10, 20], 1, { $0 * $1 })
print(a) // 1000
var languages = ["Swift", "Objective-C"]
var b = reduce(languages, "We learned: ", { $0 + $1 })
print(b) // "We learned: SwiftObjective-C"
