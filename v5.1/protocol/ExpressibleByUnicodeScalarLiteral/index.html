protocol ExpressibleByUnicodeScalarLiteral

A type that can be initialized with a string literal containing a single Unicode scalar value.

Conforming TypesExpressibleByExtendedGraphemeClusterLiteral, StaticString, Substring, Unicode.Scalar
Associated Types
associatedtype UnicodeScalarLiteralType

Valid types for UnicodeScalarLiteralType are Unicode.Scalar, Character, String, and StaticString.

The String, StaticString, Character, and Unicode.Scalar types all conform to the ExpressibleByUnicodeScalarLiteral protocol. You can initialize a variable of any of these types using a string literal that holds a single Unicode scalar.

let 単: Unicode.Scalar = "単"
print(単)
// Prints "単"
